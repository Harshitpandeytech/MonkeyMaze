# Monkey Maze

A React + TypeScript puzzle game where players help Jaggu find the minimum-cost path to the banana under time constraints.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Test

```sh
npm run test
```
